# 船舱装载优化算法模型

## 一、问题描述

在保证各物品库存消耗平衡的前提下，将船舱装载至最大容量。

---

## 二、已知参数

| 符号 | 含义 | 数据来源 |
|:-----|:-----|:---------|
| $C_w$ | 船舱重量容量 | 2000 t |
| $C_v$ | 船舱体积容量 | 2000 m³ |
| $I_i$ | 物品 $i$ 的现有库存 | 表格 |
| $D_i$ | 物品 $i$ 的每日消耗 | 表格 |
| $w_i$ | 物品 $i$ 的单位重量 (t) | 表格 |
| $v_i$ | 物品 $i$ 的单位体积 (m³) | 表格 |
| $i \in \{$GRN, NUT, MUS, DW, OVE, COF, PWO$\}$ | 物品集合 | - |

---

## 三、决策变量

| 符号 | 含义 | 类型 |
|:-----|:-----|:-----|
| $x_i$ | 物品 $i$ 的装载数量 | 整数 ≥ 0 |
| $D$ | 平衡后的库存天数（统一值） | 连续变量 |

---

## 四、目标函数

$$\max \quad Z = \max\left(\frac{\sum_i x_i w_i}{C_w}, \frac{\sum_i x_i v_i}{C_v}\right)$$

即：**最大化重量填充率和体积填充率中的较大值**（按瓶颈资源优化）

---

## 五、约束条件

### 5.1 平衡约束

$$I_i + x_i = D \cdot D_i \quad \forall i$$

即：装载后库存 = 平衡天数 × 每日消耗

### 5.2 非负约束

$$x_i \geq 0 \quad \forall i$$

### 5.3 容量约束

$$\sum_i x_i w_i \leq C_w \quad \text{(重量上限)}$$

$$\sum_i x_i v_i \leq C_v \quad \text{(体积上限)}$$

---

## 六、模型简化

将平衡约束代入，消去 $x_i$：

$$x_i = \max(0, D \cdot D_i - I_i)$$

代入目标函数：

$$\max_{D \geq 0} \quad \max\left(\frac{\sum_i \max(0, D \cdot D_i - I_i) \cdot w_i}{C_w}, \frac{\sum_i \max(0, D \cdot D_i - I_i) \cdot v_i}{C_v}\right)$$

---

## 七、求解算法

```
输入: C_w, C_v, I[], D[], w[], v[]
输出: 最优装载方案

1. 计算每种物品的初始库存天数: T_i = I_i / D_i
2. 确定平衡天数下限: D_min = max(T_i)  // 保证装载量非负
3. 搜索最优D值:
   for D = D_min to D_max step 0.01:
       weight = Σ max(0, D*D_i - I_i) * w_i
       volume = Σ max(0, D*D_i - I_i) * v_i
       fill_rate = max(weight/C_w, volume/C_v)
       记录最大值对应的D
4. 输出: 
   - 最优D值
   - 各物品装载量 x_i = max(0, D*D_i - I_i)
```

---

## 八、算法复杂度

- **时间复杂度**：$O(N \cdot M)$
  - $N$: 搜索精度（如0.01步长，约1000次迭代）
  - $M$: 物品数量（7种）
- **空间复杂度**：$O(M)$

---

## 九、Python实现

```python
import numpy as np

# 已知参数
C_w, C_v = 2000, 2000  # 船舱容量

items = ['GRN', 'NUT', 'MUS', 'DW', 'OVE', 'COF', 'PWO']
I = np.array([875, 965, 853, 224, 28, 28, 11])    # 现有库存
D = np.array([187.5, 187.5, 187.5, 44.8, 5.6, 5.6, 2.24])  # 每日消耗
w = np.array([0.9, 0.9, 0.8, 0.1, 0.02, 0.1, 0.05])  # 单位重量
v = np.array([1, 1, 1, 0.1, 0.025, 0.1, 0.05])  # 单位体积

# 计算初始库存天数
T = I / D
D_min = np.max(T)

# 搜索最优D
best_D = D_min
best_rate = 0

for D in np.arange(D_min, 20, 0.01):
    x = np.maximum(0, D * D - I)
    weight = np.sum(x * w)
    volume = np.sum(x * v)
    rate = max(weight / C_w, volume / C_v)
    
    if rate > best_rate:
        best_rate = rate
        best_D = D

# 输出结果
x = np.maximum(0, best_D * D - I)
print(f"最优平衡天数: {best_D:.2f} 天")
print(f"填充率: {best_rate*100:.2f}%")
print(f"总重量: {np.sum(x*w):.2f} t")
print(f"总体积: {np.sum(x*v):.2f} m³")
```

## 十、模型特点

| 特点 | 说明 |
|:-----|:-----|
| **单变量搜索** | 降维为一维搜索问题 |
| **线性规划** | 目标函数非线性（max函数），但可通过枚举求解 |
| **平衡优先** | 保证各物品库存天数一致 |
| **瓶颈优化** | 按重量/体积中占比较高的计算填充率 |
