# 🚀 PrunTC - 船舱装载优化器

一个专为 Prosperous Universe 设计的船舱装载优化工具，基于库存平衡算法计算最优装载方案，最大化船舱利用率。

## ✨ 功能特性

### 🎯 核心功能
- **智能优化算法**：基于二分查找的高效算法，快速找到最优平衡天数
- **多船舱支持**：8种标准船舱类型，一键切换容量配置
- **材料自动匹配**：376种材料数据库，输入代码自动填充重量/体积
- **实时计算**：即时显示装载方案、填充率和预计可用天数

### 🎨 界面特性
- **响应式设计**：适配桌面和移动设备
- **深色/浅色主题**：支持一键切换，自动保存偏好
- **玻璃拟态风格**：现代化UI设计
- **动画效果**：流畅的进度条和交互动画

### 📊 数据展示
- **填充进度条**：直观显示重量/体积利用率
- **瓶颈分析**：自动识别限制因素
- **详细清单**：每个物品的装载数量、重量、体积
- **预计可用天数**：基于消耗率计算库存持续时间

## 📖 使用指南

### 1. 选择船舱类型
- 从下拉菜单选择预设船舱类型
- 或手动输入自定义载重和容积

### 2. 添加物品
- 输入物品代码（如 `DW` `RAT` `OVE`）
- 系统自动匹配重量和体积
- 输入现有库存和每日消耗量
- 点击"添加物品"

### 3. 计算优化
- 点击"计算最优装载方案"
- 查看填充进度和详细清单
- 根据结果调整物品或容量

### 4. 管理物品
- 点击物品行删除单个物品
- 使用"清空物品"移除所有
- 使用"加载示例"快速体验

## 🧮 算法说明

### 核心规则
- **库存平衡**：所有物品补充后的库存天数相同
- **最大化填充**：优先填满重量或容积中限制较大的维度
- **整数装载**：所有装载数量必须为整数

### 优化策略
1. **二分查找**：快速收敛最优天数
2. **智能缓存**：缓存中间计算结果，避免重复
3. **提前终止**：达到99.9%填充率时自动停止
4. **边界优化**：基于容量和消耗率动态调整搜索范围

### 性能指标
- **计算速度**：平均 0.001ms/次
- **并发能力**：支持 1,000,000 次/秒
- **内存效率**：智能缓存，自动清理
