# 🚀 补给运输计算器

一个用于优化船舱装载方案的Web工具，帮助您在保证各物品库存消耗平衡的前提下，将船舱装载至最大容量。

## 功能特性

- 📦 **船舱类型选择**：支持8种不同类型的货舱
  - TCB 微型货舱 (100t/100m³)
  - VSC 超小型货舱 (250t/250m³)
  - SCB 小型货舱 (500t/500m³)
  - MCB 中型货舱 (1000t/1000m³)
  - LCB 大型货舱 (2000t/2000m³)
  - HCB 巨型货舱 (5000t/5000m³)
  - VCB 高容积货舱 (1000t/3000m³)
  - WCB 高负荷货舱 (3000t/1000m³)

- 🔍 **材料自动匹配**：输入材料代码自动获取单位重量和体积

- 🧮 **智能优化算法**：
  - 保证各物品库存消耗平衡
  - 搜索最优平衡天数
  - 计算最大填充率

- 📊 **可视化结果**：
  - 最优平衡天数
  - 填充率
  - 总重量/总体积
  - 详细装载方案表格
  - 进度条显示

## 使用方法

1. 打开 `PrunTC.html` 文件
2. 选择船舱类型（或手动输入容量）
3. 添加物品信息：
   - 物品代码（支持自动匹配）
   - 现有库存
   - 每日消耗量
4. 点击"计算最优装载方案"按钮
5. 查看优化结果

## 核心规则

- 补充后各物品的库存天数必须相同
- 按重量和体积中占用比例较高的计算填充率
- 搜索最大化填充率的装载方案

## 技术栈

- HTML5
- CSS3（响应式设计、玻璃拟态风格）
- Vanilla JavaScript（无框架依赖）
