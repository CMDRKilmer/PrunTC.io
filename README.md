# 🚀 PrunTC - 船舱装载优化器

一个专为 Prosperous Universe 设计的船舱装载优化工具，包含两个核心功能：补给运输计算器（PrunTC）和最少次数运输计算器（MTC）。

## ✨ 功能特性

### 🎯 核心功能
- **补给运输计算器（PrunTC）**：基于库存平衡算法计算最优装载方案，最大化船舱利用率
- **最少次数运输计算器（MTC）**：计算将所有物品运完所需的最少运输次数，优化运输效率
- **智能优化算法**：基于二分查找和三阶段优化策略，快速找到最优解
- **多船舱支持**：8种标准船舱类型，一键切换容量配置
- **材料自动匹配**：376种材料数据库，输入代码自动填充重量/体积
- **实时计算**：即时显示装载方案、填充率和预计可用天数

### 🎨 界面特性
- **响应式设计**：适配桌面和移动设备
- **深色/浅色主题**：支持一键切换，自动保存偏好
- **玻璃拟态风格**：现代化UI设计
- **动画效果**：流畅的进度条和交互动画
- **统一界面**：两个计算器使用一致的界面风格

### 📊 数据展示
- **填充进度条**：直观显示重量/体积利用率
- **瓶颈分析**：自动识别限制因素
- **详细清单**：每个物品的装载数量、重量、体积
- **预计可用天数**：基于消耗率计算库存持续时间（仅PrunTC）
- **运输方案**：详细的运输次数和每次装载内容（仅MTC）

## 📖 使用指南

### PrunTC - 补给运输计算器

1. **选择船舱类型**
   - 从下拉菜单选择预设船舱类型
   - 或手动输入自定义载重和容积

2. **添加物品**
   - 输入物品代码（如 `DW` `RAT` `OVE`）
   - 系统自动匹配重量和体积
   - 输入现有库存和每日消耗量
   - 点击"添加物品"

3. **计算优化**
   - 点击"计算最优装载方案"
   - 查看填充进度和详细清单
   - 根据结果调整物品或容量

4. **管理物品**
   - 点击物品行删除单个物品
   - 使用"清空物品"移除所有
   - 使用"加载示例"快速体验

### MTC - 最少次数运输计算器

1. **选择船舱类型**
   - 从下拉菜单选择预设船舱类型
   - 或手动输入自定义载重和容积

2. **添加物品**
   - 输入物品代码（如 `INS` `MCG`）
   - 系统自动匹配重量和体积
   - 输入物品数量
   - 点击"添加物品"

3. **计算优化**
   - 点击"计算最少运输次数"
   - 查看运输次数和详细方案
   - 根据结果调整物品或容量

4. **管理物品**
   - 点击物品行删除单个物品
   - 使用"重置"移除所有
   - 使用"快速示例"加载预设数据

## 🧮 算法说明

### PrunTC 算法
- **核心规则**：库存平衡（所有物品补充后的库存天数相同）
- **优化策略**：二分查找快速收敛最优天数
- **性能指标**：平均 0.001ms/次计算

### MTC 算法
- **核心规则**：最少运输次数，最大化每趟利用率
- **三阶段优化策略**：
  1. **双极限组合查找**：寻找能同时填满重量和体积的物品组合
  2. **密度互补装载**：优先装载密度最大和最小的物品
  3. **剩余空间填充**：优化剩余空间的利用
- **智能评分**：综合考虑填充率、平衡度和连续性
